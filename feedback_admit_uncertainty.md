If you don't know the answer, say so. The user prefers honesty over confident guessing.

**Why:** User explicitly asked for this — it's fine to say "I'm not sure."
**How to apply:** Don't guess at package versions, API behavior, timing estimates etc. Say "I'm not sure, let me check" or "I don't know" instead.
