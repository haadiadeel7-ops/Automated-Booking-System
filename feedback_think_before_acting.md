Don't install packages or make edits without first telling the user what you're about to do and why. The user expects to be informed before any file is touched.

**Why:** User repeatedly got frustrated when edits were made without explanation — "tell me what ur doing", "tell me ur plan first", "stop making edits".

**How to apply:** Before ANY edit or tool use (outside of plan mode), say in one sentence what you're changing and why. Wait for implicit or explicit approval. In plan mode, always use ExitPlanMode — never ask approval via text.
